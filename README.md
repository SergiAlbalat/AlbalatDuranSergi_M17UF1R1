# AlbalatDuranSergi_M17UF1R1

## Game Controls:

- A/D: Move right/left
- Space: Change gravity
- E: Next dialogue
- Esc: Pause/Unpause

## Game description:

In this little Action 2D game you control the Samurai, who has been treapped in a dungeon and needs to escape. In order to escape he have 2 powers to make things easier. First of all he can change in what direccion the gravity affects him making possible things like walk through the ceiling. His second powers give him the ability to avoid death and teleport to a past zone. This game also features the mercyless Sensei Shimura, a man who can teleport to the ceiling or the ground and can kill anyone with only touching them. Will you make the Samurai free?
